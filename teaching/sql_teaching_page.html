Learn SQL.
From first query to full mastery.

SQL is the lingua franca of data -spoken by analysts, engineers, scientists, and product managers alike. Every row in every database you'll ever touch is waiting for a query. This page is a structured, open curriculum that takes you from your first SELECT to building and securing database infrastructure at scale.

Each level comes with a video overview, per-topic exercises, and a mixed practice section that spans the whole level. Expand a level to start learning.

What is SQL?

SQL (Structured Query Language) is the standard language for talking to relational databases. You use it to ask questions of your data, transform it, and manage the tables that hold it - from a single CSV-sized table up to billion-row production systems.
