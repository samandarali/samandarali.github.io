Ridge Regression Explained

Main Formula

\[ \hat{\beta} = (X^T X + \lambda I)^{-1}X^T y \]

This is the closed-form solution for Ridge Regression.

1. Ordinary Linear Regression

In linear regression, predictions are written as:

\[ \hat{y} = X\beta \]

where:

  • X is the feature matrix
  • y is the target variable
  • β represents the coefficients

Ordinary Least Squares minimizes the sum of squared errors:

\[ RSS = (y - X\beta)^T(y - X\beta) \]

The solution is:

\[ \hat{\beta} = (X^T X)^{-1}X^Ty \]

2. The Problem with Ordinary Least Squares

OLS can become unstable when:

  • features are highly correlated,
  • there are many predictors,
  • or \(X^TX\) is close to singular.

For example, if two variables are almost identical, such as Income and Salary, the model may produce unstable coefficients:

β1 = 1450
β2 = -1447

The predictions may still be reasonable, but the coefficients become too large and unreliable.

3. Ridge Regression Idea

Ridge Regression adds a penalty for large coefficients.

Instead of minimizing only:

\[ (y-X\beta)^T(y-X\beta) \]

Ridge minimizes:

\[ (y-X\beta)^T(y-X\beta) + \lambda\beta^T\beta \]

The second term is the L2 penalty.

4. What Does \(\lambda\) Mean?

\(\lambda\) controls how strongly the model penalizes large coefficients.

Value of \(\lambda\) Meaning
\(\lambda = 0\) Ridge becomes ordinary linear regression.
Small \(\lambda\) Small penalty; coefficients shrink slightly.
Large \(\lambda\) Strong penalty; coefficients shrink strongly toward zero.

Example:

Without Ridge:
β = [8.4, -6.7, 15.2]

With Ridge:
β = [2.9, -2.4, 4.8]

5. Deriving the Ridge Formula

The Ridge objective function is:

\[ J(\beta) = (y-X\beta)^T(y-X\beta) + \lambda\beta^T\beta \]

Take the derivative with respect to \(\beta\):

\[ -2X^T(y-X\beta) + 2\lambda\beta = 0 \]

Expand:

\[ -2X^Ty + 2X^TX\beta + 2\lambda\beta = 0 \]

Simplify:

\[ X^TX\beta + \lambda\beta = X^Ty \]

Factor out \(\beta\):

\[ (X^TX + \lambda I)\beta = X^Ty \]

Finally:

\[ \hat{\beta} = (X^TX + \lambda I)^{-1}X^Ty \]

6. Why Do We Add \(\lambda I\)?

\(X^TX\) is a matrix, so we cannot add the scalar \(\lambda\) directly to it. We add \(\lambda I\), where \(I\) is the identity matrix.

\[ I = \begin{bmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 1 \end{bmatrix} \]

This increases only the diagonal values of \(X^TX\):

\[ X^TX + \lambda I = \begin{bmatrix} 4 & 2 \\ 2 & 3 \end{bmatrix} + \begin{bmatrix} 0.5 & 0 \\ 0 & 0.5 \end{bmatrix} = \begin{bmatrix} 4.5 & 2 \\ 2 & 3.5 \end{bmatrix} \]

7. Why Ridge Helps

Adding \(\lambda I\):

  • makes the matrix easier to invert,
  • reduces unstable coefficients,
  • handles multicollinearity better,
  • reduces overfitting,
  • usually improves prediction on new data.

8. Bias–Variance Tradeoff

Model Bias Variance
Ordinary Least Squares Low High
Ridge Regression Slightly higher Lower

Ridge accepts a small increase in bias in exchange for a larger reduction in variance. This often gives better results on unseen data.

9. OLS vs Ridge Summary

Ordinary Least Squares Ridge Regression
Minimizes only RSS Minimizes RSS + L2 penalty
Formula: \((X^TX)^{-1}X^Ty\) Formula: \((X^TX+\lambda I)^{-1}X^Ty\)
No coefficient penalty Penalizes large coefficients
Sensitive to multicollinearity Handles multicollinearity better
Can overfit Usually generalizes better
Important: In practice, the intercept is usually not penalized. Most software packages shrink only the feature coefficients, not the intercept.
