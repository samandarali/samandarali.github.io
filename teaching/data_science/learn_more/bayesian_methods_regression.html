Bayesian Methods and Bayesian Regression

A practical explanation of Bayesian thinking, Bayesian regression, confidence intervals versus credible intervals, when to use Bayesian methods in machine learning, and how to run Bayesian regression in Python.

1. What Are Bayesian Methods?

Bayesian methods are statistical methods based on the idea of updating beliefs using evidence. You start with an initial belief, called the prior. Then you observe data. After seeing the data, you update your belief and obtain the posterior.

Main idea:
Prior belief + observed data = updated belief

This is different from many classical statistical methods, where parameters are treated as fixed but unknown. In Bayesian methods, uncertain quantities such as regression coefficients are treated as probability distributions.

2. Bayes' Theorem

Bayesian methods are built on Bayes' theorem:

\[ P(A\mid B)=\frac{P(B\mid A)P(A)}{P(B)} \]

In words:

Term Meaning
P(A) Prior probability: what we believe before seeing new evidence.
P(B|A) Likelihood: how likely the evidence is if A is true.
P(A|B) Posterior probability: updated belief after seeing evidence.
P(B) Evidence: a normalizing value that makes probabilities valid.

3. Bayesian Regression

Bayesian regression is a regression method where the coefficients are not treated as single fixed values. Instead, each coefficient has a probability distribution.

In ordinary linear regression, we write:

\[ y = \beta_0 + \beta_1 x + \varepsilon \]

Ordinary least squares estimates one value for each coefficient, for example:

\[ \hat{\beta}_0 = 2.35, \qquad \hat{\beta}_1 = 4.18 \]

Bayesian regression says: we are uncertain about these coefficients. So instead of only estimating one value, we estimate a distribution:

\[ \beta_1 \sim \mathcal{N}(4.18, 0.25^2) \]

This means the slope is probably around 4.18, but other nearby values are also possible.

4. Bayesian Regression Formula

For regression, Bayes' theorem becomes:

\[ P(\boldsymbol{\beta}\mid D)=\frac{P(D\mid\boldsymbol{\beta})P(\boldsymbol{\beta})}{P(D)} \]
Component Meaning in Bayesian Regression
P(β) Prior distribution of the coefficients.
P(Data | β) Likelihood: how likely the observed data are for given coefficients.
P(β | Data) Posterior distribution of the coefficients after observing the data.
P(Data) Evidence or normalizing constant.

5. The Three Main Parts

Step 1: Prior

The prior represents what we believe before seeing the data.

\[ \beta_1 \sim \mathcal{N}(0, 10^2) \]

This says the coefficient is expected to be around zero, but many values are possible. The prior can be informative, weakly informative, or non-informative.

Step 2: Likelihood

The likelihood describes how likely the observed data are for a given set of coefficients. With normal regression errors:

\[ y_i \sim \mathcal{N}(\beta_0 + \beta_1 x_i, \sigma^2) \]

Step 3: Posterior

The posterior combines the prior and the data. This is the final updated distribution of the coefficients.

Prior + Likelihood → Posterior

6. Confidence Interval vs Credible Interval

Ordinary regression also gives intervals. The important difference is not whether an interval exists. The important difference is what the interval means.

Ordinary Linear Regression

Suppose ordinary least squares estimates:

\[ \hat{\beta}_1 = 5 \]

with a 95% confidence interval:

\[ (4.2,\;5.8) \]

Many people interpret this as:

There is a 95% probability that the true coefficient lies between 4.2 and 5.8.

This interpretation is incorrect in ordinary frequentist regression.

The correct frequentist interpretation is:

If we repeatedly collected new datasets and computed a 95% confidence interval each time, about 95% of those intervals would contain the true coefficient.

In ordinary regression, the coefficient is treated as fixed but unknown. The interval is random because it depends on the random sample.

Bayesian Regression

Suppose Bayesian regression estimates:

\[ \beta_1 \sim \mathcal{N}(5,\;0.4^2) \]

Its 95% credible interval is:

\[ (4.2,\;5.8) \]

Now you can say:

There is a 95% probability that the coefficient lies between 4.2 and 5.8, given the observed data and the prior.

This is because, in Bayesian statistics, the coefficient itself is treated as a random variable with a probability distribution.

The Main Difference

Ordinary Regression Bayesian Regression
Coefficient is fixed. Coefficient is random.
Interval changes from sample to sample. Coefficient distribution is updated by the data.
Confidence interval. Credible interval.
Probability statement about the procedure. Probability statement about the parameter.
Simple summary: Ordinary regression says the method works 95% of the time in repeated samples. Bayesian regression says there is a 95% probability for the parameter range, given the data and prior.

7. When Should You Use Bayesian Regression in Machine Learning?

Good Use Cases

Small datasets

Need uncertainty estimates

Prior knowledge matters

Noisy data

Sequential updating

Avoiding overfitting

Less Suitable Cases

Very large datasets

Only prediction accuracy matters

Highly nonlinear relationships

Need very fast training

No need for uncertainty

Bayesian regression is especially useful when the question is not only “What is the prediction?” but also “How uncertain is the prediction?”

8. Practical Comparison with Other Regression Models

Situation Recommended Model
Linear relationship, enough data Linear Regression
Multicollinearity Ridge Regression
Feature selection Lasso Regression
Both multicollinearity and feature selection Elastic Net
Curved relationship Polynomial Regression
Small data plus uncertainty Bayesian Regression
Complex nonlinear pattern Random Forest, Gradient Boosting, XGBoost, Neural Networks

9. Python Example: Bayesian Regression with Scikit-Learn

The easiest way to run Bayesian regression in Python is to use BayesianRidge from scikit-learn. This model behaves like ridge regression but estimates uncertainty in the coefficients using a Bayesian approach.

Complete Example

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

from sklearn.linear_model import BayesianRidge, LinearRegression
from sklearn.model_selection import train_test_split
from sklearn.metrics import mean_squared_error, r2_score

# -----------------------------
# 1. Create sample data
# -----------------------------
np.random.seed(42)

X = np.linspace(0, 10, 100).reshape(-1, 1)
noise = np.random.normal(0, 3, size=100)

# True relationship: y = 5 + 2x + noise
y = 5 + 2 * X.ravel() + noise

# -----------------------------
# 2. Split data
# -----------------------------
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.25, random_state=42
)

# -----------------------------
# 3. Fit Bayesian regression
# -----------------------------
bayes_model = BayesianRidge()
bayes_model.fit(X_train, y_train)

# -----------------------------
# 4. Predict
# -----------------------------
y_pred = bayes_model.predict(X_test)

# -----------------------------
# 5. Evaluate
# -----------------------------
rmse = np.sqrt(mean_squared_error(y_test, y_pred))
r2 = r2_score(y_test, y_pred)

print("Intercept:", bayes_model.intercept_)
print("Coefficient:", bayes_model.coef_)
print("RMSE:", rmse)
print("R-squared:", r2)

# -----------------------------
# 6. Prediction with uncertainty
# -----------------------------
X_new = np.array([[6]])

mean_prediction, std_prediction = bayes_model.predict(
    X_new, return_std=True
)

print("Prediction for X = 6:", mean_prediction[0])
print("Prediction standard deviation:", std_prediction[0])

# Approximate 95% prediction interval
lower = mean_prediction[0] - 1.96 * std_prediction[0]
upper = mean_prediction[0] + 1.96 * std_prediction[0]

print("Approximate 95% interval:", lower, upper)

What the Code Does

Step Explanation
Create data Generates artificial data with a linear relationship plus random noise.
Fit model Uses BayesianRidge() to estimate the regression relationship.
Evaluate Uses RMSE and R-squared to measure prediction performance.
Predict with uncertainty Uses return_std=True to return both the predicted mean and uncertainty.

10. Comparing Ordinary Linear Regression and Bayesian Ridge

# Fit ordinary linear regression
linear_model = LinearRegression()
linear_model.fit(X_train, y_train)

linear_pred = linear_model.predict(X_test)

print("Linear Regression coefficient:", linear_model.coef_)
print("Bayesian Ridge coefficient:", bayes_model.coef_)

print("Linear Regression RMSE:",
      np.sqrt(mean_squared_error(y_test, linear_pred)))

print("Bayesian Ridge RMSE:",
      np.sqrt(mean_squared_error(y_test, y_pred)))

In many simple datasets, ordinary linear regression and Bayesian ridge may produce similar predictions. The main extra benefit of Bayesian regression is that it gives a probabilistic view of coefficients and predictions.

11. Visualizing Prediction Uncertainty

# Sort values for a smooth plot
X_plot = np.linspace(0, 10, 200).reshape(-1, 1)

# Predict mean and standard deviation
y_mean, y_std = bayes_model.predict(X_plot, return_std=True)

plt.figure(figsize=(9, 6))
plt.scatter(X_train, y_train, label="Training data")
plt.plot(X_plot, y_mean, label="Bayesian prediction")

# 95% uncertainty band
plt.fill_between(
    X_plot.ravel(),
    y_mean - 1.96 * y_std,
    y_mean + 1.96 * y_std,
    alpha=0.2,
    label="Approx. 95% interval"
)

plt.xlabel("X")
plt.ylabel("y")
plt.title("Bayesian Regression with Prediction Uncertainty")
plt.legend()
plt.show()

The central line is the model prediction. The shaded band represents uncertainty. Wider bands mean the model is less certain about its prediction.

12. Important Warning About BayesianRidge

BayesianRidge in scikit-learn is a convenient and practical Bayesian-style regression model, but it is not the full version of Bayesian modeling that you may see in libraries such as PyMC or Stan.

For many machine learning tasks, BayesianRidge is enough. But if you want to define custom priors, complex models, hierarchical structures, or full posterior sampling, then you would use:

  • PyMC
  • Stan
  • NumPyro
  • TensorFlow Probability

13. Simple Rule of Thumb

Use Bayesian Regression when you care about uncertainty, have limited data, or want to include prior knowledge. If you only want fast prediction accuracy on a large dataset, simpler or more powerful machine learning models may be better.

14. Summary

Bayesian methods update prior beliefs using observed data. Bayesian regression applies this idea to regression by treating coefficients as probability distributions instead of fixed values.

Ordinary regression gives confidence intervals, but Bayesian regression gives credible intervals with a more direct probability interpretation. The biggest advantage of Bayesian regression is not always better prediction accuracy; it is the ability to model uncertainty in a principled way.
