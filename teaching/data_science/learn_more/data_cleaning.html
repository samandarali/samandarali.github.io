Before Modeling: Data Understanding Checklist

1. Understand the Data ▶

Ask: What does each row represent?

  • One customer
  • One transaction
  • One patient
  • One survey response

Create a data dictionary:

Variable     Meaning
CustomerID   Unique customer
Age          Customer age
Churn        Left company: Yes/No
2. Dataset Structure ▶

Check dataset size:

df.shape

Check column types and missing values:

df.info()

Questions:

  • How many rows?
  • How many columns?
  • Which variables are numeric?
  • Which variables are categorical?
3. Missing Values ▶ 
df.isna().sum()
df.isna().mean() * 100

Questions:

  • Which variables have missing values?
  • What percentage is missing?
  • Are missing values random?
  • Is an entire group missing?
4. Duplicates ▶ 
df.duplicated().sum()
df["customer_id"].duplicated().sum()

Check for exact duplicate rows and duplicate IDs.
